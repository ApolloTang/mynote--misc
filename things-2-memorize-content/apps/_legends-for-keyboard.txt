legends for keys in notes:
--------------------------

  [ctr] (c) control
  [opt] (o) option
  [cmd] (m) command
  [sft] (f) shift
  [fnc] (n) function
  [tab] (t) tab
  [ret] (r) return
  [esc] (e) escape
  [del] (d) delete
  [bsp] (b) backspace == [fnc][del]
  [spb] (p) spacebar

  [(^)] up
  [(v)] down
  [(<)] left
  [(>)] right
