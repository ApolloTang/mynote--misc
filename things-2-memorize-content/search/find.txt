Useful options:
===============

  -name {pattern}
  -type {type}
    t all
    f file
    d directories
    l symbolic

  $ find . -name '*.js'


Search n replace recursively in a dir
=====================================

  $ find . -name '*.rb' -type f \
    -exec sed -i -e 's/<pattern>/<replacement>/g' -- {} +
