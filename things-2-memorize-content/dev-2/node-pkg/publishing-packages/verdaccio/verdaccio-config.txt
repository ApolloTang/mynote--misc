

Forgot verdaccio password:
==========================

  Read Ref [1]


Verdaccio local password is saved at:
=====================================

  ~/.config/verdaccio/htpasswd

  Ref [1]


Verdaccio configuration file is located at:
===========================================

   ~/.config/verdaccio/config.yaml

  Ref [2]


Verdaccio published package is store in:
=========================================

  ~/.local/share/verdaccio/storage

  Ref [3]
