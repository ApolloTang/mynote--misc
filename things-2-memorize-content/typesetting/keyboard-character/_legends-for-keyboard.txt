legends for keys in notes:
--------------------------

  [Ctr] (c) control
  [Opt] (o) option
  [Cmd] (m) command
  [Sft] (f) shift
  [Fnc] (n) function
  [Tab] (t) tab
  [Ret] (r) return
  [Esc] (e) escape
  [Del] (d) delete
  [Bsp] (b) backspace == [fnc][del]
  [Spc] (p) spacebar

  [(^)] up
  [(v)] down
  [(<)] left
  [(>)] right
