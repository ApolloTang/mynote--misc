=========================
  installing
  starting and stopping
  using brew services
  [1][2]
=========================

  brew tap mongodb/brew

  brew install mongodb-community@4.2
  brew services start mongodb-community@4.2

[1] https://docs.mongodb.com/manual/tutorial/install-mongodb-on-os-x/
