
How to prepare file for trasfer:
-------------------------------

Prepared:
~~~~~~~~~

  Tape archive a directory using bzip, add date stamp to archived file name:

    % tar -cjf "0__$(date +%FT%H%M).tar.bz2" ~/0

  Split archive into 100M part:

    % split -b 100m 0__2020-04-29T0116.tar.bz2 "0__2020-04-29T0116.tar.bz2.part"

After transfer (received):
~~~~~~~~~~~~~~~~~~~~~~~~~~

  Join parts:

    % cat 0--2020-04-29T0116.tar.bz2.parta* >0.tar.bz2

  unarchived:

    % tar -xjf 0.tar.bz2
