====================
Command line editing
====================

  <C-b>/<C-e>        to begin/end of line
  <S-[→]>/<S-[←]>    move cursor by word
  <C-w>              delete a word
  <C-u>              delete full line


=======================
Command line Completion
=======================

  <C-l><C-P>        auto complete full keyword forward/back
  <C-d>             list all matched completable


===========
More Tricks
===========

  <C-R><C-W>   : Pull <cword> onto search(/) or command(:) line
