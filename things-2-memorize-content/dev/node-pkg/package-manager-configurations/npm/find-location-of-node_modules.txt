::: Find out what is the root folder for current diretory

        npm root

    the result of above command is where npm will consider
    as node_module in pwd so it different depend on where you
    issue this command.


----------------------------------------------------------

Use the following command to show where npm install global packages:

  $ npm root -g

Note that if you use nvm to manage your node version, the global packages,
is install under the vesrion of node:

  $ npm root -g
  /Users/<USER>/.nvm/versions/node/v14.17.3/lib/node_modules

And in various plateform:

  windows:
  C:\Users\<USERNAME>\AppData\Roaming\npm\node_modules

  vagrant box
  /usr/local/lib/node_modules

  mac
  /usr/local/lib/node_modules
