
List installed
``````````````
  nvm list


list all available
``````````````````
  nvm ls-remote


Install a specified version
```````````````````````````
  nvm install 14.4.1


Apply a specified version
`````````````````````````
  nvm use 14.4.1


Specified a default version
````````````````````````````
  nvm alias default 14.4.1


update to latest lts node
`````````````````````````
  nvm install node --latest-npm --reinstall-packages-from=node

  [1] https://github.com/nvm-sh/nvm/issues/1706
