Initializer
===========
  $ npm i -g create-react-app
  $ create-react-app my-app

  $ npx create-react-app my-app
  $ npm init react-app my-app
  $ yarn create react-app my-app


Get version list
================
  $ npm info react time
  $ yarn info react time


Get a list package
==================
  $ yarn list --depth 0
  $ npm list --depth 0
