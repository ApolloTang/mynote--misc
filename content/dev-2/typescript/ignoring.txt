This is how you ts-ignore a jsx property
========================================

    /*
    // @ts-ignore */
    id="foo does not have IdType"
    baz="foo Have bazType"
  />

Work around to ignore TSX error in jsx property
===============================================

  -  <CardMedia
  -    component="img"
  -    src={imageUrl}
       ^^^----------- tsc error in 'src' property
  -    className={classes.media}
  -    title="foo"
  -  />

  +  {React.createElement(CardMedia, {
  +    // @ts-ignore
  +    src: imageUrl,
  +    component: 'img',
  +    className: classes.media,
  +  })}

  ref: https://stackoverflow.com/a/40245490/3136861
