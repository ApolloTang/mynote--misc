
Useful
======

  -----
    ping -w 2 8.8.8.8    # run a ping test for two seconds and report
                         # on the network statistics gathered.

    Note:
      8.8.8.8 is a public DNS service that is provided by Google [1]
      Option -w is not available on OSX

  -----

    ping on OSX and linus behave differently. [2]
